# Grid Padel Tournament Generator

A desktop application for managing padel tournament brackets with double-elimination format.

## Features
- Up to 32 pairs of players
- Double-elimination: Main Way + Consolation Way
- BYE support for odd numbers of pairs
- Visual bracket rendering with PDF export
- Auto-advancement on result entry

## Tech Stack
- Java 21+ / Spring Boot 3.x / JavaFX
- Maven build system
- Clean Architecture + DDD + TDD

## Building
```bash
mvn clean package
```

## Running
```bash
mvn javafx:run
```
